Analog Signal Conditioning Interface

This project presents the design and simulation of an analog signal conditioning interface composed of several cascaded stages. The system processes an input signal through amplification, filtering, programmable gain control, and rectification.

The circuit includes four main stages:

Instrumentation Amplifier – Amplifies the input signal while maintaining high accuracy and low distortion.

Analog Low-Pass Filter – Limits the bandwidth and removes high-frequency noise.

Programmable Gain Amplifier (PGA) – Allows adjustable amplification using switchable gain configurations.

Precision Rectifier – Converts the AC signal into a DC representation of the signal amplitude.

The project includes circuit design, component sizing, and simulations such as DC operating point, AC analysis, transient response, bandwidth measurement, slew rate evaluation, and THD analysis.

Simulations and validation were performed to verify that the circuit meets the required performance specifications.
